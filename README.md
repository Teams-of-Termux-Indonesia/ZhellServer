<h2>instalasi terminal linux</h2>

``apt update && apt upgrade;
pgk update && pkg upgrade;
pkg install git;
git clone https://github.com/ryugenxd/ZhellServer;
cd ZhellServer;
chmod 777 install;
./install;``
