### step installation

```shell
# update && upgrade package
apt update && apt upgrade || pkg update && pkg upgrade
```

```shell
# install git package
pkg install git -y
```

```shell
# clone this project
git clone https://github.com/ryugenxd/ZhellServer && cd ZhellServer
```

```shell
# run this project
chmod 777 install && ./install
```
